## 送信先
audiosdk@avid.com

## 件名
AAX Commercial License Request - ZeroLimit Plugin Beta Testing

## 本文

```
Dear Avid AAX Team,

I am developing an audio plugin called "ZeroLimit" and would like to request access to the AAX commercial license and PACE signing tools.

## Project Details

**Plugin Name:** ZeroLimit
**Version:** 0.1.0
**Plugin Type:** Zero-latency broadcast brickwall limiter
**Formats:** VST3, AU, AAX, Standalone
**Target Market:** Broadcast engineers, post-production engineers, mastering engineers

**Description:**
ZeroLimit is a zero-latency brickwall limiter designed to be inserted at the final stage of a DAW for broadcast and live production use. It runs with no lookahead (0 sample latency) and exposes a minimal control surface (Threshold, Output Gain) alongside input level, gain reduction, and output level meters.

## Reason for Commercial License Request

I need to obtain the AAX signing tools to conduct beta testing with professional audio engineers. Several professional engineers have agreed to test the plugin, but they cannot install Pro Tools Developer Edition on their production workstations due to:

1. Company IT policies that restrict software installation
2. Risk of disrupting their daily commercial work environment
3. Need to test the plugin in real-world production scenarios with their existing Pro Tools setup

Therefore, I need to sign the AAX plugin so it can run on standard Pro Tools installations for beta testing purposes.

## Development Status

- Development: early-stage, minimum DSP and UI working
- Current testing: In-house testing with Pro Tools Developer Edition
- Next phase: Beta testing with broadcast/mastering engineers
- Expected commercial release: TBD
```

## メール送信時のポイント

1. **具体的な理由を明確に記載**
   - プロのエンジニアがDeveloper Editionをインストールできない理由
   - ベータテストの重要性

2. **プロジェクトの真剣度を示す**
   - 開発がほぼ完了していること
   - 具体的なテスター（プロのエンジニア）がいること

3. **必要な情報を全て含める**
   - プラグインの詳細
   - 開発者情報
   - 配布計画

4. **iLokの準備状況を明記**
   - すでにアカウントがある、またはすぐに作成する意思があること

## 返信の目安

- 通常1-3営業日で初回返信
- 承認まで1週間程度
- 承認後、PACEからも自動的に連絡が来る

## 補足

「commercial license」という名前ですが、実際には「プロダクション環境でのテスト」という正当な理由があれば、問題なく承認される可能性が高いです。
